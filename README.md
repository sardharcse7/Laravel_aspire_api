# Laravel_aspire_api

Step:1
Download the project in your sustem by using link, https://github.com/sardharcse7/aspireapi
cd laravel-passport-aspire-loan-api

Step2:
Configure your database from .env file
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravel
DB_USERNAME=root
DB_PASSWORD=

Step:3
composer install

Step:4
php artisan migrate

step:5
php artisan serve
